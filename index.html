Підготовка

Швиденько повторюємо

Усе, як на минулому уроці: браузер → trinket.io/turtle → видаляємо стандартний приклад. Готові писати своє.

main.py
тут має бути порожньо (виділив усе → Delete)
поки порожньо —
картина з'явиться згодом
Не забудь /turtle Адреса має бути саме trinket.io/turtle, а не trinket.io/python3 — інакше turtle не запрацює.
Крок 1

Знайомий каркас

Імпорти, черепаха, чорне полотно. Майже все знаємо — тільки два нових рядки: hideturtle() і penup().

main.py
import turtle import random t = turtle.Turtle() t.speed(0) t.hideturtle() t.penup() screen = turtle.Screen() screen.bgcolor("black")
чорне полотно,
ще без малюнка
Що роблять нові рядки

hideturtle() — ховає стрілочку черепахи. На цьому уроці вона нам не потрібна.

penup() — піднімає «олівець». Коли черепаха переміщатиметься, слід (лінії) не залишатимется. Ми малюємо кружечки, а не лінії.

Крок 2

Перший кружечок

Команда t.dot(розмір) малює заповнений круг там, де зараз черепаха. На старті це центр екрана — точка (0, 0).

main.py
# ...каркас той самий... screen.bgcolor("black") t.color("red") t.dot(40)
Спробуй різні розміри t.dot(10) — крапочка. t.dot(100) — велике коло на півекрана. Кольори: "red", "yellow", "#FF00AA".
Крок 3

Стрибаємо в інше місце

Щоб поставити кружечок не в центрі — спочатку треба перемістити черепаху. Це робить t.goto(x, y). Завдяки penup() жодних ліній за нею не лишається.

main.py
# ...каркас... t.color("red") t.dot(40) t.goto(100, 120) t.color("yellow") t.dot(40)
Координати у turtle
У turtle вгору — це +y. Точка (100, 120) опиниться правіше та вище центру.
Крок 4

Випадкова позиція

Головна ідея уроку. Замість того, щоб самим думати, куди поставити кружечок, попросимо комп'ютер вибрати координати навмання. Це робить random.randint(від, до).

main.py
# ...каркас... x = random.randint(-200, 200) y = random.randint(-200, 200) t.goto(x, y) t.color("red") t.dot(40)
Як працює randint(-200, 200) Це функція з бібліотеки random. Вона повертає випадкове ціле число від −200 до 200 включно. Може випасти −187, 0, 42, 199 — будь-що. Кожен виклик — нове число.
Запусти кілька разів Натисни ▶ Run п'ять разів підряд. Кружечок з'являтиметься у різних місцях — комп'ютер щоразу обирає інші x та y.
Крок 5

І розмір випадковий

Той самий прийом для розміру: randint(10, 50) — випадкове число від 10 до 50. Маленькі цяточки і великі диски — впереміш.

main.py
# ...каркас... x = random.randint(-200, 200) y = random.randint(-200, 200) size = random.randint(10, 50) t.goto(x, y) t.color("red") t.dot(size)
Спробуй ширший діапазон randint(5, 100) — від крапки до півекрана. Картина стане ще різноманітніша.
Крок 6

Випадковий колір

Цей прийом ми вже знаємо з минулого уроку: список кольорів і random.choice, що бере з нього один навмання.

main.py
# ...каркас... colors = ["red", "yellow", "blue", "green", "orange", "cyan"] x = random.randint(-200, 200) y = random.randint(-200, 200) size = random.randint(10, 50) t.goto(x, y) t.color(random.choice(colors)) t.dot(size)
Розбір random.choice(colors) colors — список (у квадратних дужках, через кому). choice бере один елемент зі списку випадково.
Крок 7

Цикл — і кружечків стає 50

Тепер усе разом: загортаємо вибір координат, розміру, кольору і малювання — у цикл на 50 повторень. Кожен оберт — новий випадковий кружечок.

main.py · фінальний код
import turtle import random t = turtle.Turtle() t.speed(0) t.hideturtle() t.penup() screen = turtle.Screen() screen.bgcolor("black") colors = ["red", "yellow", "blue", "green", "orange", "cyan"] for i in range(50): x = random.randint(-200, 200) y = random.randint(-200, 200) size = random.randint(10, 50) t.goto(x, y) t.color(random.choice(colors)) t.dot(size)
Слідкуй за відступами Все, що всередині циклу, має відступ 4 пробіли. Python без відступів не зрозуміє, що ці рядки треба повторювати.
Готово

Картина готова — і вона унікальна

Натисни ▶ Run кілька разів. Щоразу — нова картина: інші позиції, інші розміри, інші кольори. Цього точного малюнка більше ніхто й ніколи не побачить.

Завдання 1

Експерименти з числами

Поміняй одне число у коді — і отримай зовсім іншу картину.

A

200 замість 50

range(50) → range(200). Кружечків стане вчетверо більше — буде як зорі на нічному небі.

B

Тільки великі

randint(10, 50) → randint(50, 150). Дрібнички зникають — лишаються гігантські планети.

C

Тільки вгорі

В рядку для y: randint(-200, 200) → randint(0, 200). Картина займе тільки верхню половину.

Бонус для швидких

Сузір'я з ниток

Що буде, якщо опустити олівець? Тоді при переміщенні черепаха залишатиме за собою лінії — і випадкові точки з'єднаються у сузір'я.

Зміни в коді
main.py
# каркас БЕЗ t.penup() colors = ["white", "yellow", "cyan"] for i in range(30): x = random.randint(-200, 200) y = random.randint(-200, 200) t.color(random.choice(colors)) t.goto(x, y) t.dot(8)

Прибери рядок t.penup() з каркасу — і одразу goto почне залишати лінії між точками.

Що з'явиться
Ще трюки t.pensize(2) на початку — нитки товщі. range(80) замість 30 — буде павутина.
